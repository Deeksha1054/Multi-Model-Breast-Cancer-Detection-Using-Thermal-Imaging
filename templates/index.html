About Our System

Our AI-powered breast cancer detection system uses state-of-the-art deep learning technology to analyze thermal imaging data and identify potential abnormalities with remarkable accuracy.

How It Works

The system employs a two-stage detection pipeline:

  • Stage 1: Initial screening using ResNet-18 architecture trained on thermal RGB images to classify tissue as NORMAL or potentially concerning (SICK)
  • Stage 2: For cases flagged as concerning, a secondary model performs detailed analysis to distinguish between benign and malignant abnormalities
  • Real-time Processing: Results are generated in seconds, providing immediate feedback for healthcare providers

Technology Stack

Built with cutting-edge machine learning frameworks:

  • PyTorch deep learning framework for model training and inference
  • ResNet-18 convolutional neural network architecture
  • Flask web framework for seamless user interaction
  • Thermal imaging analysis for non-invasive screening

Our Mission
